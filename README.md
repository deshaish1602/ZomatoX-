# 🍽️ ZomatoX – Restaurant Data Analysis

## 📖 Overview

ZomatoX is a data analysis project that explores restaurant datasets to uncover insights related to pricing, ratings, cuisines, and customer preferences.
The project focuses on **Exploratory Data Analysis (EDA)** to identify patterns and trends that can help in business decision-making.

---
## 🌐 Live Demo

  https://vduzbtyqgbe8svbv7ipcff.streamlit.app

 


## 🚀 Features

* 📊 Exploratory Data Analysis (EDA)
* ⭐ Rating distribution analysis
* 💰 Price range insights
* 🍜 Cuisine popularity trends
* 🌍 Location-based analysis
* 📈 Data visualization using graphs and charts

## 📊 Key Insights

* High-rated restaurants are not always expensive
* Most ratings are clustered between **3.5 – 4.5**
* Certain cuisines dominate specific regions
* Price distribution varies significantly by location

## ▶️ How to Run Locally

1. Clone the repository
   git clone https://github.com/deshaish1602/ZomatoX-

2. Navigate to the folder
   cd ZomatoX-

3. Install dependencies
   pip install -r requirements.txt

4. Run the notebook
   jupyter notebook

5. (Optional) Run Streamlit app
   streamlit run app.py
